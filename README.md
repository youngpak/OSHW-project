# OSHW-project

오픈소스하드웨어 개인 프로젝트 과제

주제: 학교 공지사항 어플

선정 이유: 매일 같이 공지사항을 확인하기 위해 학교 홈페이지, IT융합대학 홈페이지, SW사업단 홈페이지를 차례대로 돌아가며 확인한다. 
          이 번거로운 절차를 앱 하나로 간편하게 해결 할 수 있는 서비스를 만들고자 한다.
